Let it snow on your WordPress website with lots of fun options!

== Description ==

It's the most wonderful time of the year, but... what about the snow? This plugin got you covered.

This plugin will add snow to your website. With more than 14 customizable options, you'll be garanteed to get into the Christmas spirit!

== Installation ==

1. Upload the 'snow' folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Snow to configure the plugin
